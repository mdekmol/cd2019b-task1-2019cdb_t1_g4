在git add .    提交推送的時候，運到了下圖問題

在上網查詢之後，發現是因為不同平台上換行符被修改時，Git會認為整個文件修改，

不過Git也有提供指令解決問題

#提交時轉換為LF，check out時不轉換
git config --global core.autocrlf   input

# 拒絕提交包含混合換行符的文件
git config --global core.safecrlf  true
 
設定完check out   .gifconfig



之後就可以正常推送
